# Amabi — Dashboard Fintech

Tela responsiva de dashboard desenvolvida para o projeto **Fintech Amabi**, como atividade acadêmica da FIAP.

## Sobre o projeto

A Amabi é uma interface de controle financeiro pessoal. A tela desenvolvida apresenta uma visão geral do usuário, com saldo disponível, entradas, saídas, evolução do saldo e navegação lateral.

## Tecnologias utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilos complementares
- **Tailwind CSS via CDN** — layout, grid, espaçamentos e responsividade
- **Font Awesome via CDN** — ícones da interface
